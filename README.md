# CartShare 

## Collaborative Shopping Cart Web Application

### About the Project

CartShare is a collaborative shopping cart web application developed as part of my Web Development Internship Project. The application helps groups such as roommates, friends, or office teams manage a shared shopping list efficiently.

Users can create or join a shopping room using a unique room code and collaboratively add, edit, or remove items from a shared cart. The application also maintains an activity log, tracks room members, calculates the total bill automatically, and generates a printable receipt.

---

## Features

- User login using SessionStorage
- Create and join shopping rooms
- Unique room code generation
- Shared shopping cart management
- Add, edit, and delete shopping items
- Item categorization and automatic total calculation
- Activity log for tracking user actions
- Member management
- Real-time synchronization across browser tabs using the Storage Event API
- Responsive design for desktop, tablet, and mobile devices
- Printable shopping receipt

## How to Run the Project

1. Download or clone this repository.
2. Open the project folder.
3. Launch `index.html` in any modern web browser.
4. Enter your username.
5. Create a new room or join an existing room using a room code.
6. Start adding items and collaborate with other users.
